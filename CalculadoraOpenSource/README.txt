1. 📁 ESTRUCTURA DEL PROYECTO
 estructura de carpetas:

CalculadoraOpenSource/

├── Project/
   ├── main.py   ← Aquí va el menú principal
   ├── dividir.py
   ├── multiplicacion.py
   ├── resta.py
   ├── sumar.py
   └── suma_avanzada.py

2. ▶️ CÓMO EJECUTAR EL PROYECTO


Paso 1: Ejecuta el archivo principal:

     main.py

4. 🧪 FUNCIONAMIENTO DEL MENÚ

El programa mostrará las siguientes opciones:

    1 - Suma de dos números
    2 - Resta de dos números
    3 - Multiplicación de dos números
    4 - División de dos números
    5 - Suma de varios números
    6 - Salir

Ingresa el número de la acción que deseas realizar.

5. ⚠️ VALIDACIONES IMPORTANTES

- Asegúrate de ingresar solo números válidos cuando el programa lo solicite.
- Si introduces letras o símbolos, el sistema te pedirá que vuelvas a intentarlo.
- Si eliges una opción inválida del menú, se mostrará un mensaje de error.
